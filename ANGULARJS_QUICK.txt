ðŸ’» CARA KERJA
================================================================

1. LOAD DATA (List Controller):
   
   ApiService.siswa.getAll()
       .then(function(response) {
           $scope.siswas = response.data.data;
       });

2. CREATE DATA (Create Controller):
   
   $scope.formData = { nama: 'Ahmad', ... };
   
   ApiService.siswa.create($scope.formData)
       .then(function(response) {
           AlertService.success('Berhasil ditambahkan');
           $location.path('/siswa');
       });

3. UPDATE DATA (Edit Controller):
   
   var id = $routeParams.id;
   
   // Load data
   ApiService.siswa.getById(id)
       .then(function(response) {
           $scope.formData = response.data.data;
       });
   
   // Submit update
   ApiService.siswa.update(id, $scope.formData)
       .then(function(response) {
           AlertService.success('Berhasil diupdate');
       });

4. DELETE DATA (List Controller):
   
   $scope.deleteSiswa = function(siswa) {
       if (!confirm('Yakin hapus?')) return;
       
       ApiService.siswa.delete(siswa.id)
           .then(function(response) {
               $scope.loadData(); // Reload
           });
   };

================================================================
ðŸ› TROUBLESHOOTING
================================================================

Problem: Halaman blank
Solution: 
  - Check browser console (F12)
  - Verify all JS files loaded
  - Check module name matches ng-app

Problem: Data tidak muncul
Solution:
  - Test API: http://localhost/siswa/public/api/siswa
  - Check browser console for errors
  - Verify CORS configuration

Problem: Form submit tidak jalan
Solution:
  - Check ng-submit="submitForm()"
  - Verify submitForm() exists in controller
  - Check console for errors

Problem: CORS error
Solution:
  - Set 'allowed_origins' => ['*'] in cors.php
  - Restart Laravel server
  - Clear browser cache
