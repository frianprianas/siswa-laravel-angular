ğŸ› TROUBLESHOOTING
================================================================

Problem: Menu tidak active
Fix: Cek request()->routeIs('namaroute.*')

Problem: Layout tidak muncul
Fix: Pastikan @extends('layouts.app') di awal file

Problem: Hamburger tidak kerja
Fix: Cek Bootstrap JS loaded

Problem: Error 404
Fix: php artisan route:list (cek routes)

Problem: CSRF token mismatch
Fix: @csrf di form, clear browser cache

Problem: Data tidak muncul
Fix: Cek controller return view dengan data
