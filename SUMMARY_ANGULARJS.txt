ğŸ’¡ KONSEP PENTING:
================================================================

1. SPA (Single Page Application)
   - Hanya load HTML once
   - Navigation tanpa reload
   - Data via AJAX/API

2. REST API
   - Laravel sebagai backend
   - Return JSON format
   - Standard HTTP methods (GET, POST, PUT, DELETE)

3. Two-Way Data Binding
   - ng-model bind input ke variable
   - Perubahan otomatis update view
   - Contoh: <input ng-model="nama"> â†’ {{ nama }}

4. Promises
   - Async operation handling
   - .then() untuk success
   - .catch() untuk error

5. Dependency Injection
   - Angular inject service ke controller
   - Contoh: ['$scope', 'ApiService', function($scope, ApiService) {}]

Problem: Halaman blank

  - Buka Console (F12), cek error
