Tentang Aplikasi

Aplikasi Siswa - Sistem Informasi Manajemen

Aplikasi manajemen data siswa yang dibangun menggunakan Laravel dan AngularJS. Sistem ini dirancang untuk memudahkan pengelolaan data siswa, kelas, dan jurusan dengan interface yang modern dan user-friendly.

Teknologi

Laravel 8.x & AngularJS 1.8

Database

MySQL

UI Framework

Bootstrap 5 & Font Awesome

Pengembang

By Frian

Fitur Aplikasi
  • Manajemen Data Siswa - CRUD lengkap untuk data siswa
  • Manajemen Kelas & Jurusan - Kelola data kelas dan jurusan
  • Import CSV - Upload data siswa secara masal dengan validasi
  • Export ke Excel - Download data dalam format CSV/Excel
  • Filter Data - Filter siswa berdasarkan kelas dan jurusan
  • Dashboard - Statistik dan ringkasan data real-time
  • Authentication - Sistem login dengan keamanan token
  • Dark Theme - Mode gelap untuk kenyamanan mata
  • Responsive Design - Tampilan optimal di semua perangkat
